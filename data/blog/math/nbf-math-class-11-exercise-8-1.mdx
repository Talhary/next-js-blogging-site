# Exercise

### 1. Find the values of **cos(α ± β)**, **sin(α ± β)**, **tan(α ± β)** for each given pair of angles.

---

#### (i) α = 180°, β = 60°

**Solution:**

For **cos(α + β):**

```math
\cos(α + β) = \cos(180° + 60°) = \cos(240°) = -\cos(60°) = -\frac{1}{2}
```

For **sin(α + β):**

```math
\sin(α + β) = \sin(180° + 60°) = \sin(240°) = -\sin(60°) = -\frac{\sqrt{3}}{2}
```

For **tan(α + β):**

```math
\tan(α + β) = \tan(180° + 60°) = \tan(240°) = \tan(60°) = \sqrt{3}
```

---

#### (ii) α = -60°, β = 90°

**Solution:**

For **cos(α + β):**

```math
\cos(α + β) = \cos(-60° + 90°) = \cos(30°) = \frac{\sqrt{3}}{2}
```

For **cos(α - β):**

```math
\cos(α - β) = \cos(-60° - 90°) = \cos(-150°) = -\cos(30°) = -\frac{\sqrt{3}}{2}
```

For **sin(α + β):**

```math
\sin(α + β) = \sin(-60° + 90°) = \sin(30°) = \frac{1}{2}
```

For **sin(α - β):**

```math
\sin(α - β) = \sin(-60° - 90°) = -\sin(30°) = -\frac{1}{2}
```

For **tan(α + β):**

```math
\tan(α + β) = \tan(-60° + 90°) = \tan(30°) = \frac{1}{\sqrt{3}}
```

---

#### (iii) α = 180°, β = 30°

**Solution:**

For **cos(α + β):**

```math
\cos(α + β) = \cos(180° + 30°) = \cos(210°) = -\cos(30°) = -\frac{\sqrt{3}}{2}
```

For **sin(α + β):**

```math
\sin(α + β) = \sin(180° + 30°) = \sin(210°) = -\sin(30°) = -\frac{1}{2}
```

For **tan(α + β):**

```math
\tan(α + β) = \tan(180° + 30°) = \tan(210°) = \tan(30°) = \frac{1}{\sqrt{3}}
```

#### (iv) α = π, β = 2π/3

**Solution:**

For **cos(α + β):**

```math
\cos(α + β) = \cos(π + \frac{2π}{3}) = -\frac{1}{2}
```

For **sin(α + β):**

```math
\sin(α + β) = \sin(π + \frac{2π}{3}) = -\sin(\frac{2π}{3}) = -\frac{\sqrt{3}}{2}
```

For **tan(α + β):**

```math
\tan(α + β) = \tan(π + \frac{2π}{3}) = \tan(\frac{2π}{3}) = -\sqrt{3}
```

For **tan(α - β):**

```math
\tan(α - β) = \tan(π - \frac{2π}{3}) = -\tan(\frac{π}{3}) = -\sqrt{3}
```

---

#### (v) α = 4π/3, β = π/6

**Solution:**

For **cos(α + β):**

```math
\cos(α + β) = \cos(\frac{4π}{3} + \frac{π}{6})
             = \cos(\frac{5π}{3})
             = \cos(\frac{2π}{3} - \frac{π}{3})
             = \frac{1}{2}
```

For **cos(α - β):**

```math
\cos(α - β) = \cos(\frac{4π}{3} - \frac{π}{6})
             = \cos(\frac{7π}{6})
             = \cos(π + \frac{π}{6})
             = -\cos(\frac{π}{6}) = -\frac{\sqrt{3}}{2}
```

For **sin(α + β):**

```math
\sin(α + β) = \sin(\frac{4π}{3} + \frac{π}{6})
             = \sin(\frac{5π}{3})
             = -\sin(\frac{\pi}{3}) = -\frac{\sqrt{3}}{2}
```

For **sin(α - β):**

```math
\sin(α - β) = \sin(\frac{4π}{3} - \frac{π}{6})
             = \sin(\frac{7π}{6})
             = -\sin(\frac{π}{6}) = -\frac{1}{2}
```

For **tan(α + β):**

```math
\tan(α + β) = \frac{\sin(α + β)}{\cos(α + β)}
             = \frac{-\frac{\sqrt{3}}{2}}{\frac{1}{2}}
             = -\sqrt{3}
```

For **tan(α - β):**

```math
\tan(α - β) = \frac{\sin(α - β)}{\cos(α - β)}
             = \frac{-\frac{1}{2}}{-\frac{\sqrt{3}}{2}}
             = \frac{1}{\sqrt{3}}
```

&\text{Given:} \\
&\sin(\alpha - \beta) = \sin\left(-\frac{\pi}{2}\right) = -1 \\
&\sin(\alpha + \beta) = \sin\left(\frac{2\pi}{3}\right) = \frac{\sqrt{3}}{2} \\
&\cos(\alpha + \beta) = \cos\left(\frac{2\pi}{3}\right) = -\frac{1}{2} \\
&\tan(\alpha - \beta) = \tan\left(-\frac{\pi}{2}\right) = \text{undefined} \\
&\tan(\alpha + \beta) = \tan\left(\frac{2\pi}{3}\right) = -\sqrt{3} \\

&\text{Let } \alpha = \frac{4\pi}{3}, \beta = \frac{\pi}{6} \\

&\text{Solution:} \\
&\cos(\alpha + \beta) = \cos\left(\frac{4\pi}{3} + \frac{\pi}{6}\right) \\
&= \cos\left(\frac{4\pi}{3}\right)\cos\left(\frac{\pi}{6}\right) - \sin\left(\frac{4\pi}{3}\right)\sin\left(\frac{\pi}{6}\right) \\
&= \left(-\frac{1}{2}\right)\left(\frac{\sqrt{3}}{2}\right) - \left(-\frac{\sqrt{3}}{2}\right)\left(\frac{1}{2}\right) \\
&= -\frac{\sqrt{3}}{4} + \frac{\sqrt{3}}{4} \\
&= 0 \\

&\cos(\alpha - \beta) = \cos\left(\frac{4\pi}{3} - \frac{\pi}{6}\right) \\
&= \cos\left(\frac{4\pi}{3}\right)\cos\left(\frac{\pi}{6}\right) + \sin\left(\frac{4\pi}{3}\right)\sin\left(\frac{\pi}{6}\right) \\
&= \left(-\frac{1}{2}\right)\left(\frac{\sqrt{3}}{2}\right) + \left(-\frac{\sqrt{3}}{2}\right)\left(\frac{1}{2}\right) \\
&= -\frac{\sqrt{3}}{4} - \frac{\sqrt{3}}{4} \\
&= -\frac{\sqrt{3}}{2} \\

&\text{As } \tan(\alpha - \beta) = \frac{\sin(\alpha - \beta)}{\cos(\alpha - \beta)} \\
&= \frac{-1}{-\frac{\sqrt{3}}{2}} = \frac{2}{\sqrt{3}} \\
\end{align*}
